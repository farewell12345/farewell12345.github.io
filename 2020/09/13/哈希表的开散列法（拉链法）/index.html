哈希表的开散列法（拉链法）

Posted on 2020-09-13
672- 2m-
数据结构Java

当我们需要进行大批量数据查询或搜索时需要用到散列存储来加快搜索速度，但是散列比较容易发生冲突，因而为了解决散列存储带来的冲突问题，我们可以用到哈希表的开散列法（拉链法）来解决这个问题。

对于产生地址冲突的数据，将溢出的数据添加到该元素对应的链表里。 以搜索平均长度为的链表代替了搜索长度为 n 的顺序表，搜索效率快的多。

拉链法的基本思路如下：

构建一个关键码子集，里面存放关键码，每个关键码都对应一个链表，类似这种：

这张图里的关键码子集是{0,1,2,3,4,5,6}，而其后的链表用来存放对应的数据。看起来就像一个个拉链，因为这种方法也叫 拉链法

在添加数据的时候，通过计算该数据对应的关键码（通过设计好的哈希函数计算）将该元素添加到对应关键码链表的末端。这样就实现了一个简单的哈希表的开散列法。

节点结构：

public class Node {
    private int key;//数值
    public Node next;//next指针

    public int getKey() {
        return key;
    }

    public void setKey(int key) {
        this.key = key;
    }
}
public class Hash<T> {
    private static int HashMax=10;
    //关键码子集的最大容量
    private static Node[] hashTable=new Node[HashMax];
    //创建关键码子集

    void setHashMax(int value){
        HashMax=value;
        //修改关键码子集
    }

    public static int HashMod(int key){
        return key % HashMax;
        //取模计算哈希关键码
    }

    //插入新的元素
    public static void InsertHashTable(int key){
        int index=HashMod(key);
        Node t=hashTable[index];
        //是否存在该元素对应的关键码
        if (t!=null) {
            //存在则将该元素添加到关键码对应的链表末端
            AddHashTable(key);
        } else {
            //不存在就创建该关键码并将该元素设定为该关键码的第一个元素
            t=new Node();
            hashTable[index]=t;
            t.setKey(key);
        }
    }

    //将某元素添加到关键码对应的链表末端
    public static void AddHashTable(int key){
        Node tNode=hashTable[HashMod(key)];
        //获取对应关键码
        while (tNode.next!=null){
                tNode=tNode.next;
        }
        tNode.next=new Node();
        tNode.next.setKey(key);
    }
    //打印整张哈希表
    public static void printHashTable(){
        for (Node i: hashTable) {
            Node j=i;
            while (j!=null){
                System.out.print(j.getKey());
                System.out.print("\t");
                j=j.next;
            }
            System.out.println();
        }
    }
    //根据key值搜索某个节点
    public static Node HashSearch(int key){
        Node p;
        int index=HashMod(key);
        p=hashTable[index];
        while (p != null){
            if (p.getKey()==key){
                return p;
            }
            p=p.next;
        }
        throw new NullPointerException("不存在这个值");
    }
}
