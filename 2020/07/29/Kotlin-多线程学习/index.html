Kotlin-多线程学习

Posted on 2020-07-29- Edited on 2020-07-31
216- 1m-
kotlin

Kotlin的多线程类似于Java，可以通过封装Java里的Thread类或调用Runnable接口实现多线程操作。

但是需要注意的是，Kotlin中没有 synchronized等关键字。

封装Thread类：

重写run函数

object :Thread() {
        override fun run() {
            println("I am Thread")
        }
    }.start()

使用Lambda表达式

Thread(){
        println("I am Thread ")
    }.start()

封装thread()函数

import kotlin.concurrent.thread
/*thread函数所接受的参数别分为：
是否直接运行线程（start）
是否作为守护线程（isDaemon),
线程名字（name）
线程优先级（priority)*/
val t = thread(start = true, isDaemon = false, name = "thread", priority = 3) {
        println("thread")
    }

同步锁

kotlin中没有Synchronized关键字，而是将其转变为了标签

@Synchronized
fun Fun2()
{
    print("I am new Thread")
}

实例化Runnable接口

class newRunnable():Runnable{
    override fun run() {
        print("Runnable!")
    }
}

val a=newRunnable()
a.run()

使用Lambda表达式

val a=Runnable{
        print("Runnable!")
}
a.run()
